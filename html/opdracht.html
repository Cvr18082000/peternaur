De Opdracht

We hadden de opdracht gekregen om een site te maken met behulp van HTML en CSS. De site moest gaan over een recente winnaar van de Turing Award. Voordat we zijn begonnen maakten we een plan van aanpak. Hierin stond hoe we de site gingen maken, hoe deze eruit ging zien, de taakverdeling en wat voor informatie er op de site zou komen. Ook hebben we opdrachten gemaakt en instructievideo's bekeken op de site Khan Academy. Dit is een site waarop je stap voor stap leert hoe je met HTML en CSS werkt. Daarnaast hebben we natuurlijk de opdrachten van Informatica-Actief gedaan. Met deze kennis konden we gaan beginnen aan de HTML van deze site. De opmaak van deze site is gedaan met CSS. Dat hebben we gebruikt om de site mooi te maken, en alles op de goede plek te zetten. Toen we klaar waren met de site hebben we hem online gezet met GitHub. Dit is een site waar je bestanden kunt uploaden en kan bewerken samen met anderen.

Wat is HTML?

HTML staat voor HyperText Markup Language. Het is een programmeertaal die we voornamelijk gebruiken om een webpagina op te stellen. Het is ontwikkeld door het W3C, het World Wide Web Consortium. HTML bestaat echter alleen uit platte tekst, dus tekst zonder opmaak. Vaak wordt CSS gebruikt om de tekst en de pagina op te maken.
Een voorbeeld van een stukje html: 

<!DOCTYPE HTML>
   <html>
      <head>
         <title>Paginatitel</title>
      </head>
      <body>
         <p>Hallo!</p>
      </body>
   </html>

Wat is CSS?

Css staat voor Cascading Style Sheets. CSS wordt gebruikt om webpagina's op te maken. Met CSS kun je niet alleen tekst en plaatjes opmaken maar je kunt ook de elementen op de goede plek zetten in de site. Je kunt een klein stukje HTML met oneindig veel manieren opmaken en rangschikken. In deze site gebruiken we ook CSS. Dat zorgt er bijvoorbeeld voor dat de achtergrondkleur blauw is.
Een voorbeeld van een stukje CSS: 
p{
   font-size: 16px
   color: blue
}

body{
   background-color: green
}
